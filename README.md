# Crossword Puzzle Project

This project is a simple HTML5 crossword puzzle that focuses on art-related terms, specifically "puntillismo," "perspectiva," "punto de horizonte," "artística," and "sombreado."

## Project Structure

```
crucigrama-html5
├── src
│   ├── index.html         # Main HTML document for the crossword puzzle
│   ├── styles
│   │   └── main.css       # Styles for the crossword puzzle
│   ├── scripts
│   │   └── main.js        # JavaScript logic for the crossword puzzle
│   └── data
│       └── clues.json     # Clues for the crossword puzzle in JSON format
├── package.json           # Configuration file for npm
├── .gitignore             # Files and directories to ignore by Git
└── README.md              # Documentation for the project
```

## Getting Started

To set up the project, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd crucigrama-html5
   ```

2. **Install dependencies** (if any):
   ```bash
   npm install
   ```

3. **Open the project**:
   Open `src/index.html` in your web browser to view the crossword puzzle.

## Usage

- The crossword puzzle will display a grid with clues related to the specified art terms.
- Users can fill in the answers based on the clues provided.
- JavaScript handles the logic for checking answers and managing user interactions.
